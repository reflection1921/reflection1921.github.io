About KakaoStory Enhanced

KakaoStory Enhanced는 웹 카카오스토리를 조금 더 편리하게 사용할 수 있도록 개선해주는 확장 스크립트 입니다.

* 지원 브라우저: Chrome, Safari, Firefox, Whale, Edge 등 Tampermonkey 및 기타 호환 스크립트 도구를 지원하는 브라우저(IE 미지원)

Enhanced를 사용하면 아래와 같은 기능을 사용할 수 있습니다.
- Discord 스타일의 다크 테마 외 4종의 다크테마 지원
- 폰트 및 폰트 크기 설정 가능(로컬 폰트 및 CSS 활용한 폰트 지원)
- 알림 기능 지원
- 상대방이 댓글에서 자신을 언급했을 경우 Discord의 Mention 스타일처럼 표시
- 강화된 차단 기능(차단한 사용자의 댓글을 안보이게 가려주는 기능)
- 문자열 차단 기능(특정 문자열이 들어간 글을 안보이게 가려주는 기능)
- 친구목록 백업 기능
- 기타 기능: 텔러/채널 버튼 가리기, 추천 친구 숨기기, 과거의 오늘 있었던 추억들, 이모티콘 크기 설정, 동영상 다운로드 기능

Install KakaoStory Enhanced
Enhanced for Web KakaoStory
※Tampermonkey 및 Chrome 브라우저를 기준으로 설명합니다. 호환 스크립트 도구 및 해당 도구를 지원하는 브라우저에서도 설치하실 수 있습니다.

1. 여기를 클릭해서 Tampermonkey를 설치합니다.

2. 상단의 Install Script를 클릭합니다.

3. Tampermonkey에서 스크립트 설치 창이 표시되면 설치를 클릭하여 설치합니다.
