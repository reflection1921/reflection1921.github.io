웹 카카오스토리를 즐기면서
직접 알림센터 클릭 없이 PC 알림에서 확인해보세요!

아래에 보여지는 특징은 Windows 기준입니다.
macOS용 앱은 아래의 특징에 없는 기능이 있을 수도, 아래의 특징에 있는 기능이 없을 수도 있습니다.
macOS용 앱의 개발은 CenoX님이 진행하고 있으며, 해당 앱의 저작권 역시 CenoX님께 있습니다.
  • 쉬운 사용
  • 알림 확인
  • 윈도우 7 이상
  • macOS 지원

간편한 UI

기본 기능은 하나의 윈도우 안에서 사용할 수 있도록 구성하였습니다.

윈도우 알림센터에서 알림 확인

스토리 알림센터에 알림이 오면 윈도우 알림센터에서 알림을 알려드립니다.
알림을 더블클릭하시면 알림과 관련한 게시글로 이동됩니다.
(윈도우 10 미만에서는 풍선 알림으로 대체되어 표시됩니다.)

로그 기록

알림 시작을 누르는 순간부터 누구에게 어떤 알림이 언제 왔는지 로그에 기록합니다.
(이 기능은 선택사항입니다.)
