V 1.0.4 업데이트 내역
1.카카오 로그인 오류 수정
 -카카오 로그인이 되지 않는 오류를 수정하였습니다.

2.캡챠 로그인 삭제
 -더 이상 캡챠 로그인을 지원하지 않아 삭제하였습니다. 

V 1.0.3 업데이트 내역
1.카카오 API 레벨 조정
 -카카오 API 레벨을 조정하여 몇가지 버그를 수정하였습니다.

V 1.0.2 업데이트 내역

1.버그 수정
 -로그인 및 캡챠 Form에서 어떤 키를 눌러도 다음 동작을 수행하는 버그가 발견되어 해당 부분의 코드를 제거하였습니다.
  해당 부분은 작동하지 않을 것 입니다.

V 1.0.1 업데이트 내역

1.업데이트 Form 추가
 -새로운 버전 알림이 표시 될 경우 업데이트 내역에 대해 보여주는 Form이 추가됩니다.

2.영상 저화질 다운로드 옵션 추가
 -저화질을 선택하여 저화질 영상을 다운로드 할 수 있는 옵션이 추가됩니다.

3.편의 개선
 -페이지 검색시 다운로드 경로를 설정하지 않았을 경우 자동으로 설정 탭으로 이동합니다.
 -로그인 및 캡챠 Form에서 엔터키 사용이 가능해졌습니다.

4.업데이트 알림 OFF
 -프로그램이 있는 디렉터리에 NO_UPDATE라는 이름의 텍스트파일(.TXT)을 생성하면 업데이트 알림을 띄우지 않습니다.
